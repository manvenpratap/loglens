Quick Guide

1. Import Log4j XML — Use the Log4j tab to import your app config. LogLens auto-converts appender patterns to regex templates, giving you the exact line format your app writes.

2. Add Rules — Guided 🧙 — Pick your log format, type keyword markers (e.g. TX-BEGIN), choose what to capture. LogLens generates the regex for you.

3. Add Rules — Advanced ⚙ — Full regex control with a click-to-insert token palette and live color-coded match preview (each capture group highlighted in a distinct color).

4. Stack Behaviors

  • push Opens a timed block
  • pop Closes last block & calculates duration
  • inline Standalone point event

5. Hotspots — Tree nodes show ↑↑80% badges — % of parent time consumed — instantly revealing bottlenecks.

Keyboard Shortcuts

  • Alt+T — Toggle theme
  • Alt+[/] — Prev / next thread
  • Alt+1/2/3 — Gantt / Tree / Split view
  • j / k — Navigate tree nodes
  • / — Search tree by name
  • p — Pin focused tree node
  • Esc — Close modal / sheet / menu
